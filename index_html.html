Welcome to Feel++

Feel++ is a C++ library for partial differential equation solves using generalized Galerkin methods i.e. fem, hp/fem, spectral methods.

Feel++ is currently used in various projects hosted by Cemosis. This includes

  • Fluid structure interaction in blood flows
  • High field magnets simulation
  • Optical tomography
  • Aerodynamics and aerothermal flows
  • Scalable preconditioners
